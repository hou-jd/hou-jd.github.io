  • Coursework 1: Serial Message Decoding (Individual Project, Nov 2023)

    This project focused on interpreting binary-encoded telemetry data sent from a simulated robotic system. Using Python, I implemented a decoder that extracted multi-format sensor values including GPS, orientation, and system status from raw byte streams. The project emphasized accuracy in bit manipulation, real-time parsing, and protocol adherence under constraints.

    My Contribution: I independently developed the message decoding algorithm by reading protocol layouts, used binary slicing techniques, applied struct-like unpacking logic, and thoroughly tested decoding accuracy on sample data files provided. The implementation included debugging tools and flexible structure for additional frame types.

    Skills Improved: Binary operations, Python scripting, bitwise logic, protocol-driven design, and data visualization.

  • Coursework 2: A* Path Planning Algorithm for Mobile Robots (Individual Project, Mar 2024)

    In this assignment, I implemented the A* path planning algorithm within a GUI framework to allow a mobile robot to autonomously find optimal paths while avoiding obstacles. The project strictly limited motion to 4 directions (up/down/left/right) and used Euclidean heuristics to estimate cost-to-go, all done using native Python without external libraries.

    My Contribution: I implemented the core A* logic, defined node structures, cost metrics (g, h, f), and optimized path reconstruction. The final solution was fully integrated into the GUI provided, allowing live interaction by users. I also added debugging messages and fail-safe mechanisms for handling invalid grid inputs.

    Skills Improved: Graph search algorithms, algorithmic efficiency, grid-based robotics planning, real-time GUI integration, and Python algorithm design.
